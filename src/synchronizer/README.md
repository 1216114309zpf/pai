## Synchronizer


### Environmental Varaibales Accepted by Containers

All the following variables are read by `core/config.js`:

  - `RBAC_IN_CLUSTER`
  - `K8S_APISERVER_URI`
  - `K8S_APISERVER_CA_FILE`
  - `K8S_APISERVER_TOKEN_FILE`
  - `K8S_TIMEOUT_SECONDS`
  - `DB_CONNECTION_STR`
  - `DB_MAX_CONNECTION`
  - `LIST_INTERVAL_SECONDS`
